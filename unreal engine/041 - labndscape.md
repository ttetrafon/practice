# Landscape

## Normal Landscape

- Links:
  - _[Landscape Technical Guide](https://docs.unrealengine.com/5.0/en-US/landscape-technical-guide-in-unreal-engine/)_
- To create and/or edit a landscape, select landscape mode from the toolbar.
- **Landmass details -> Erosion Settings -> Simulate Erosion** can be used to erode the landscape, making it more realistic.
  - **Launch Erosion Timeline** opens a window where the process above can be finetuned.

## Mesh Terrain

- Layers:
  - `BrushModifier`
  - `BooleanModifier`
  - `NoiseModifier`
  - `RemeshModifier`
- Adaptive Polygon Count
- Custom Tools for mesh terrain can be added with BP and PCG
