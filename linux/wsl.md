# Windows Subsystem for Linux

## Installation

- Install Kali from windows store
- Add all default tools & packages (`sudo apt install -y kali-linux-large`)

### Win Kex

- Provides a GUI desktop for Kali in WSL.
  - [Docs](https://www.kali.org/docs/wsl/win-kex/)
- Install with `sudo apt install -y kali-win-kex`
- Run with:
  - `kex --win -s` (window mode)
  - `kex --esm --ip -s` (enhanced session mode)
  - `kex --sl -s` (seamless mode)
