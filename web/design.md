# Web Design

## Colours

- [Colour Hunt](https://colorhunt.co/)
- Use a 4-colour palette for design.
  - primary background (60%): the lightest/most neutral colour
  - base colour (30%): medium-toned coloured, used for backgrounds of special elements/sections (sidebars, card backgrounds, secondary buttons, etc)
  - action (10%): the boldest/brightest colour (primary buttons, links, icons, highlights, etc)
  - typography: the darkest colour (headlines, body text, borders)
- Colour brightness should be inverted in the above for a dark theme.
